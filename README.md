## 🔥 Tentang

Projek ini adalah starter kit web modern yang fokus pada performa, kemudahan pengembangan, dan pengalaman pengguna yang baik. Cocok untuk prototipe, landing page, dan aplikasi frontend sederhana.

Dibangun menggunakan:
- JavaScript (vanilla / modular)
- HTML5
- CSS3 (dengan struktur yang rapi untuk skalabilitas)

## ✨ Fitur Utama

- Struktur proyek terorganisir
- Mudah dikembangkan dan dipelihara
- Optimasi performa dasar (struktur untuk minify/bundling)
- Template halaman start/landing
- Panduan kontribusi dan workflow dasar

## ⚙️ Persyaratan

Pastikan sudah terpasang:
- Node.js v14+ (untuk tooling)
- npm atau pnpm/yarn

## 🛠️ Instalasi & Menjalankan Secara Lokal

1. Clone repo

```bash
git clone https://github.com/Masdo-Official/projek.git
cd projek
```

2. Install dependensi (jika ada)

```bash
npm install
# atau
# yarn
# pnpm install
```

3. Jalankan server pengembangan

```bash
npm run dev
# atau sesuai script yang tersedia
```

Buka http://localhost:3000 atau port yang ditampilkan di terminal.

## 📌 Tips & Best Practices

- Gunakan modul JS untuk menjaga scope dan menghindari global pollution.
- Pisahkan style ke file terpisah dan gunakan naming convention (BEM atau lainnya) untuk skalabilitas.
- Minimalkan asset dan gambar untuk mempercepat loading.
